ATTENTION! Please read the instructions in full to
there were no further questions about the use.

The program is a text notepad capable of working with
text documents.

Menu:

The "Edit" and "Format" buttons do not work in this version. Instead of them
Select text and right-click on it.

With unsaved changes to the text in the menu near the file header
the "*" icon appears, indicating that the changes made to the file
not yet saved.

ВНИМАНИЕ! Прошу прочитать инструкцию полностью, дабы

Программа является текстовым блокнотом, способным работать с

Меню:

Кнопки "Edit" и "Format" в данной версии не работают. Вместо них
следует выделять текст и нажимать по нему правой кнопкой мыши.

При несохранённых изменениях текста в меню около заголовка файла
появляется значок "*", означающий, что внесённые изменения в файле
пока что не сохранены.
